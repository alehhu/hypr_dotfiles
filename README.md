# ❄️ My Hyprland Dotfiles

A portable, macOS-inspired Hyprland setup with dynamic Pywal integration and a "Pro" Quick Action workflow.

## 🚀 Quick Start (on a new machine)

1. **Clone the repo**:
   ```bash
   git clone <your-repo-url> ~/dotfiles
   cd ~/dotfiles
   ```

2. **Run the installer**:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

## ⌨️ Keybindings Highlights

- **`SUPER + Space`**: Search Apps (Rofi)
- **`SUPER + A`**: Quick Action Center (Lab, Power, Translate, OCR)
- **`SUPER + H`**: Searchable Keybindings List
- **`SUPER + Q`**: Close Window (macOS style)
- **`SUPER + SHIFT + 3/4`**: Screenshots
- **`SUPER + CTRL + Space`**: Emoji Picker

## 📦 What's Included?

- **Window Manager**: Hyprland (Snappy animations, gaps)
- **Bar**: Waybar (Dynamic modules)
- **Launcher**: Rofi (Custom theme)
- **Notifications**: SwayNC (Blurred, dynamic colors)
- **Shell**: Starship + Zsh aliases
- **Theming**: Pywal (Changes everything based on wallpaper)

## 🛠️ Update your dotfiles
Whenever you change something in `~/.config`, run:
```bash
./setup.sh
```
Then commit and push your changes!
